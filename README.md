# 寰域 whola-zone

新时代的分享社区
> 不要吝啬知识的分享，只要你学习、进步速度够快，就没有人能够追得上你。


**标签**
-  一份职业，一个领域，一种编程语言，一种技术框架，万事万物皆有标签来定义，类似维度、向量的定义  。

- 一切事物都可以用多个标签来“定义”，就像说起水果会想到苹果，想到红富士（品种），想到洛川、静宁（产地），想到陕西、甘肃（省），思维就这样发散出去；想到甘肃，想到兰州，碰巧我有一个朋友在兰州，思维回到我自己身上。  

**探索**
- 参考推荐算法，从一个标签出发，关联到另一个标签，如今大数据推荐，越来越精准，越来越深入你心。面对扑面而来的信息，你是否也会迷茫，阅读不过来，甚至觉得这些被推荐来的信息，占据了太多你的关注。
- 很多时候，想要了解怎么样的信息，或许你心里并没有一个明确的方向；在标签和标签的关联下，不断探索，知道某个事情引起你持续的关注；它可以使爱好，可以是提升自我的知识、技能，也可以是寻找志同道合的朋友之法门。

# 寰
- 所有用户都在同一个网络，同一数据库，同一服务器，开怀畅聊，发表自己的看法，记录自己的生活，分享学习的知识。
- 标签的产生、发展、留存，由这个世界所有的用户参与。
# 域
- 你有自己的圈子，或者你想在某些固定的话题内，认识朋友，交流心得，选择你认可的标签，组成专属于你的**域**
