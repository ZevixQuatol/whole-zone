# 寰域 HuanYu

新时代的分享社区，共同参与，共同管理
> 不要吝啬知识的分享，只要你学习、进步速度够快，就没有人能够追得上你。


**寰点**
- 一份职业，一个领域，一种编程语言，一种技术框架，万事万物皆可以由寰点定义。寰点不是完整语义本身，而是社区可见、可治理、可聚合的语义颗粒。

- 一切事物都可以用多个寰点来“定义”，就像说起水果会想到苹果，想到红富士（品种），想到洛川、静宁（产地），想到陕西、甘肃（省），思维就这样发散出去；想到甘肃，想到兰州，碰巧我有一个朋友在兰州，思维回到我自己身上。
- 寰点负责显式表达，向量负责隐式关系。文章与寰点、用户与寰点、用户与文章之间更细腻的语义距离，交由向量空间表达。

**探索**
- 参考推荐算法，从一个寰点出发，关联到另一个寰点，如今大数据推荐，越来越精准，越来越深入你心。面对扑面而来的信息，你是否也会迷茫，阅读不过来，甚至觉得这些被推荐来的信息，占据了太多你的关注。
- 很多时候，想要了解怎么样的信息，或许你心里并没有一个明确的方向；在寰点和寰点的关联下，不断探索，直到某个事情引起你持续的关注；它可以是爱好，可以是提升自我的知识、技能，也可以是寻找志同道合的朋友之法门。
- 显式寰点帮助用户理解内容归属，隐式向量帮助系统发现用户未明确表达但已经产生的潜在兴趣。

# 寰
- 所有用户都在同一个网络，同一数据库，同一服务器，开怀畅聊，发表自己的看法，记录自己的生活，分享学习的知识。
- 寰点的产生、发展、留存，由这个世界所有的用户参与。
# 域
- 你有自己的圈子，或者你想在某些固定的话题内，认识朋友，交流心得，选择你认可的寰点，组成专属于你的**域**
# 构建
- 起初，网站没有寰点，用户发布内容时，系统根据文章标题和内容智能分析、向量召回候选寰点，用户确认后建立文章与寰点关系；用户也可自定义增删改查寰点，发布后寰点进入当前域的寰点池。
- 系统同时为文章生成向量，并维护寰点向量、用户兴趣向量。寰点用于社区治理与聚合，向量用于计算文章与寰点、用户与寰点、用户与文章的隐式关系。
# 发展
- 在寰或者某一个单一的域内，用户阅读他人发布的文章内容，此时会发生什么：
```
1. 当前用户关联文章的寰点，寰点粘性+1。
2. 用户给文章寰点评级，或是视频对寰点的喜爱程度，赞同喜爱的寰点，粘性+1；不感兴趣寰点粘性-1；评级由低到高可分为：屏蔽、厌恶、不喜、无感、见识、有趣、认可、钟爱。
3. 用户想要持续关注作者的动态，可以专注文章作者，作者发布新内容将会推送至用户；用户可能只想关注作者某一些寰点的持续动态，在订阅时，选择作者的寰点则只推送对应寰点的内容。
```
- 寰点会越来越多，越来越乱，此时用户可能也不知道应该如何探索，才能寻找到自己想要的内容怎么办：
```
1. 服务器定期对寰点评级，分别统计 7 天、10 天、30 天每个寰点的热度，对于热度低于一定标准的寰点，系统进行回收，不再显示在寰点池中。
2. 同样的，对于寰点与寰点之间的关联度，用户也可以评级，同样根据一定周期进行评级、回收处理。
```
# 延申
- 用户阅览的内容越多，涉及到的寰点就越多；重复类型的内容越多，某些寰点的粘性就会越高，这样用户的寰点画像自然而然就形成了。
- 用户与用户之间，就可以通过这些寰点，寻找到自己志同道合的朋友；
- 根据用户寰点的粘性，系统推送的内容也就越来越精准，按需推送。
# 流动
- 每当用户接触到新的寰点时，会对这个寰点所关联的域进行匹配，当用户与某个域中的寰点完全重合时，用户则自动成为这个域的核心成员，这样用户就从一个域冲入到另一个域。
- 已经属于某个域的核心成员可以邀请其他用户，被邀请用户直接进入域，但是与当前域的寰点粘性并不高，所以判断用户是否为域的核心成员的方式为：寰点匹配 100% 并且粘性达到所有域用户粘性程度的 80%。
- 用户可以创建专属自己的域
1. 域只能为开放状态，所有用户可见，所有用户可进入，选择专属的寰点。
2. 域的存在与否，同样是由系统判断：根据活跃用户数量，以及域中寰点与用户粘性程度，若持续 100 天均较低，则回收域。
3. 域的管理员可以对域中的寰点进行增删，管理员不可直接删除域的用户，若存在异常用户（广告，非法传播等），提交至官方系统统一处理。
4. 官方认定的从事广告，刻意破坏等无道德行为进行最高无限期封禁；对于存在违法行为的进行永久封禁，并将注册信息计入黑名单池，不可再用相关信息再次进行注册。
